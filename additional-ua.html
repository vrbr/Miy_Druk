Вимоги до макетів

  • Цифровий друк

    Для цифрового другу

    • Формат файлів: векторні (CDR,PDF), растрові (TIFF, JPEG)
    • Колірна модель: CMYK
    • Размір файлу - не більше 50 Мб
    • Размір файлу повинен відповідати друкованому відбитку (1:1)
    • Для JPEG-файлів не використовувати стиснення
    • Для TIFF-файлів - звести всі шари в один і при збереженні вибирати ZIP або LZW стиснення
    • Для PDF-файлів - всі шрифти повинні бути впроваджені в PDF
    • Роздільна здатність: 300 dpi
    • Макет не повинен містити міток обріза, штампів, біговок і будь-яких інших недрукованих об'єктів

  • Широкоформатний друк

    Для широкоформатного друку

    • Формат файлів: TIFF, EPS
    • Растрова графіка в натуральному розмірі (1:1)
    • Роздільна здатність: 100 dpi
    • Колірна модель: CMYK
    • Чорний колір повинен бути композитним: C-60% M-60% Y-60% K-100%
